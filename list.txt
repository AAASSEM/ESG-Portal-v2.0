  Overview

  The generateEmissionsData() function processes existing meter data to calculate greenhouse gas emissions across three scopes, using
  UAE-specific emission factors.

  Step-by-Step Process:

  1. Data Source

  - Uses existing chart data (no separate API call needed)
  - Processes chartData.dataEntries which contains all meter readings
  - Each entry has: task.element_name, task.value, task.submission, etc.

  2. Scope Classification

  The getScopeForMeterType() function categorizes each meter type:

  üî¥ Scope 1 - Direct Emissions:
  - LPG, Petrol, Diesel, Natural Gas, Refrigerants
  - Examples: Fuel generators, vehicles, gas systems

  üîµ Scope 2 - Indirect Energy Emissions:
  - Electricity, Energy, Renewable Energy, District Cooling
  - Examples: Grid electricity, cooling systems

  ‚ö´ Scope 3 - Other Indirect Emissions:
  - Water, Waste (default category)
  - Examples: Water treatment, waste disposal

  3. Emission Factors (UAE-Specific)

  The getEmissionFactor() function applies conversion factors:

  Scope 1 Factors:
  - LPG: 0.003 kg CO2e/kg
  - Petrol: 0.00231 kg CO2e/L
  - Diesel: 0.00268 kg CO2e/L
  - Natural Gas: 0.002 kg CO2e/unit
  - Refrigerants: 1.3 GWP factor

  Scope 2 Factors:
  - Electricity: 0.00023 kg CO2e/kWh (UAE grid average)
  - District Cooling: 0.00009 kg CO2e/RT-h

  Scope 3 Factors:
  - Water: 0.0005 kg CO2e/m¬≥ (treatment)
  - Waste: 0.00006 kg CO2e/kg (landfill)

  4. Calculation Formula

  For each meter reading:
  Emissions (tCO2e) = Value √ó Emission Factor

  Example Calculation:
  - Electricity: 1,000 kWh √ó 0.00023 = 0.23 tCO2e
  - Water: 500 m¬≥ √ó 0.0005 = 0.25 tCO2e

  5. Data Flow

  1. Extract meter readings from chart data
  2. Classify each reading into Scope 1, 2, or 3
  3. Apply appropriate emission factor
  4. Calculate emissions for each scope
  5. Aggregate totals by scope
  6. Display in pie chart with breakdown
